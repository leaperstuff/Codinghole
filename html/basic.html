Introduction to HTML

HTML stands for HyperText Markup Language. It is used to structure web pages.

HTML consists of elements represented by tags such as <html>, <head>, and <body>.

HTML Tags and Closing

HTML tags define elements. Some tags require closing, while others are self-closing.

  • Opening and Closing: <p>Hello</p>
  • Self-Closing: <br>, <img src="image.jpg" />
