HTML Buttons

Buttons in HTML allow users to interact with your webpage by performing actions.

A simple button is created using the <button> tag.

<button>Click Me</button>

Types of Buttons

HTML provides different types of buttons:

  • Submit Button: Used in forms to submit data.
    • <button type="submit">Submit</button>
  • Reset Button: Resets form inputs.
    • <button type="reset">Reset</button>
  • Button Type: Used for JavaScript interactions.
    • <button type="button">Click Me</button>
